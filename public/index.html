📖 Dashboard Help

1. Timestamp Parsing

Timestamps are parsed in M/D/YYYY H:mm:ss format (e.g., 2/26/2026 9:15:15). By default, timestamps are interpreted in Asia/Kolkata (IST) timezone. Use the timezone dropdown in the header to convert display to another timezone. Rows with missing or invalid timestamps are ignored from charts but shown in the table.

2. Date Presets

Use the quick presets in the filter panel: Today, This Week, This Month, This Year, or All Time. Select Custom to pick any date range using the date inputs.

3. Exporting Data

Click the Export CSV or Export XLSX button at the bottom of the page to download the currently filtered dataset. The export includes all visible columns.
